The Usage of JavaScript - Interacting with HTML

The past lessons have demonstrated some basic concepts of JavaScript

However JavaScript is rarely used to pop-up little alert boxes. It is used to dynamically create content, respond to user interaction, order lists, and more.

rivers.run is a website I created (No Internet Explorer Support Yet). It uses JavaScript for searching, sorting, drawing graphs, fetching the latest river heights, expanding the drop-downs when users click, and caching itself for offline use.

The following lessons will teach you how to use JavaScript to create HTML elements, delete HTML elements, style HTML elements, and modify the content of HTML elements - All using JavaScript
